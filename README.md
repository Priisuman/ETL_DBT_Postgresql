# Weather ETL: dbt & PostgreSQL Pipeline

This project demonstrates a local Medallion Architecture ETL pipeline. It ingests raw user/weather data, processes it through a transformation layer using dbt, and outputs a clean 'Gold' schema in PostgreSQL.

## 🛠 Tech Stack
- **Database:** PostgreSQL 
- **Transformation:** dbt (data build tool)
- **Language:** Python 3.13 (Virtual Environment)
- **Automation:** Windows Task Scheduler & Batch Scripting

## 🏗 Architecture
1. **Landing Schema:** Raw data ingested via Python as JSON/Text.
2. **Gold Schema:** Cleaned, flattened, and type-casted tables generated by dbt.

## 🚀 How to Run
1. Activate environment: `.\dbt_stable_env\Scripts\activate`
2. Run dbt: `dbt run`
3. Test data: `dbt test`
